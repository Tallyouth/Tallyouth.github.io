个人总结

  • 框架，很多公司都会强调精通一门JavaScript框架，只要一门精通就行，足够了，但是，精通指的是你了解他的运行机制，他的优点缺点，从最简单的项目搭建开始一直到项目提交打包，都要知道流程，即使你不会用到。
    什么时候算精通呢，再我看来，就是你在使用他的时候，感觉这个地方不合理，你想改他，你就去百度，发现都没这方面的相关东西。在别的框架才有。

  • JavaScript，比如只懂 jQuery 不懂 JavaScript 是不行的哦，JavaScript是一门语言，熟练掌握作用域、闭包、执行模型、性能优化、内存管理、网络通信、缓存等。很多人会问你es6新特新，他只是JavaScript的一种新的版本，你经常用的时候，就会理解，除非你一直写jQuery，那你就真的只是个前端。记住，要为了一门语言学框架，而不是为了前端要写框架。js精通可以转nodejs，会vue你转什么？

  • 知识面，记住，要不广，要不深。
